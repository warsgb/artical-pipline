# 原始素材

> Multi-Agent Dev Team v2.2 - 一个框架，搭建任意类型的 AI 协作团队

## 核心价值定位

Multi-Agent Dev Team不是一个"开发团队脚手架"，而是一个通用的多AI代理协作框架。它通过交互式向导，帮你在 OpenClaw 上快速搭建任意类型的专业团队——无论是软件开发团队、投资分析团队，还是你自己定义的任何团队。

## 解决的问题

1. **角色单一化问题**：让同一个 AI 代理又写代码、又做架构、又测试，质量悄悄下滑
2. **多项目并行需求**：同时在做多个项目，需要多个独立团队并行运作
3. **垂直领域限制**：想组建投资分析团队，但那不是"开发团队"，没有现成工具
4. **重复配置成本**：每次搭建新团队都要手动改配置文件，反复踩同样的坑

## 真实案例验证

### 团队一���Coding Team（7 人软件开发团队）
📋 PM → 🏗️ 架构师 → 🎨 前端 + ⚙️ 后端（并行）→ 🛠️ Code Artisan（代码审查）→ 🔍 QA → 🚀 DevOps

完整的软件交付流水线。每个角色都有独立的工作空间、专属的系统提示、适配的模型。

### 团队二：Wealth Team（4 人投资分析团队）
🎯 CIO（总指挥）→ 📊 Researcher（宏观+行业研究）→ 🛡️ Risk（风控+压力测试）→ ⚡ Trader（交易执行计划）

每日盘前晨会、并行研究、独立风控的金融团队。同样用这个框架搭建，同样稳定运行。

**本质**：框架是通用的，团队是你定义的。

## 支持的团队类型

**技术类**
- 软件开发团队（PM + 架构 + 前后端 + QA + DevOps）
- 数据工程团队（数据工程师 + 分析师 + 安全 + 文档）
- 安全审计团队（渗透测试 + 合规 + 文档）

**业务类**
- 投资分析团队（CIO + 研究员 + 风控 + 交易）
- 内容创作团队（策划 + 写手 + 编辑 + 分发）
- 市场调研团队（研究 + 分析 + 报告撰写）

**自定义**
- 任何你能想到的角色组合，2 到 10 人，完全由你决定

## 核心能力

### 灵活的团队规模
2 到 10 个代理，按需组合。不同团队可以有完全不同的规模和结构。

### 多团队并行运行
一个 OpenClaw 实例可以同时运行多个独立团队。

## 核心亮点
- **通用框架设计**：不是垂直领域工具，而是可定义任意团队的框架
- **真实案例验证**：已有Coding Team和Wealth Team两个成功案例
- **角色专业化**：每个AI代理专注单一角色，避免质量下滑
- **交互式向导**：降低搭建门槛，快速部署
- **多团队并行**：支持同时运行多个独立团队
