# Claude Prompt：灵感入库

## 使用场景
当你看到一个好内容（文章、视频、金句、热点），想把它存进内容工厂并生成选题。

## Prompt 模板

```
@用Claude和Obsidian做内容生产工厂.md

帮我把这条灵感存进内容工厂：

【灵感内容】
"""
[这里粘贴你的灵感：可以是链接、截图文字、复制的内容]
"""

请执行以下操作：

1. 在 `01-灵感与素材库/1-日常灵感剪报` 创建灵感卡
   - 标题格式：日期+关键词，如「2024-03-07-AI写作方法论」
   - 使用模板：[[90-系统配置/模板/灵感模板]]
   - 填写完整YAML frontmatter

2. 基于这个灵感，生成3个可写的选题，保存到 `02-选题池/待写选题库/`
   - 每个选题用单独文件
   - 使用选题模板
   - 选题要具体、有爆款潜力、有差异化角度

3. 自动建立回链：
   - 灵感卡链接到3个选题
   - 每个选题链接回灵感卡

完成后告诉我：
- 创建的文件路径
- 3个选题的标题和核心角度
```

## 简化版（快速使用）

如果你已经很熟悉，可以直接说：

```
把这条存进内容工厂，并生成3个选题：

[粘贴灵感内容]
```

---

## 示例

**输入：**
```
把这条存进内容工厂，并生成3个选题：

看到一篇文章说：「真正的效率不是做得更多，而是少做但必须做的事」
```

**Claude 会：**
1. 在 `01-灵感与素材库/1-日常灵感剪报/` 创建「2024-03-07-少即是多的效率观.md」
2. 在 `02-选题池/待写选题库/` 创建3个选题文件，如：
   - 「2024-03-07-为什么越忙的人效率越低.md」
   - 「2024-03-07-极简工作法的3个原则.md」
   - 「2024-03-07-从 todo list 到 not-to-do list.md」
3. 所有文件自动建立回链
