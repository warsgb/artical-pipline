# Claude Prompt：大纲转内容

## 使用场景
当你确定了要写的选题和大纲，让 Claude 帮你生成完整的初稿内容。

## Prompt 模板

```
@用Claude和Obsidian做内容生产工厂.md

帮我把这个大纲写成完整的文章。

【大纲来源】
- 文件：{{大纲文件路径}}
- 或粘贴大纲内容：
"""
[粘贴大纲内容]
"""

【输出要求】
- 平台：{{公众号/小红书/X/抖音脚本等}}
- 字数：{{大概字数要求}}
- 语气：{{专业/亲切/犀利/温暖}}
- 特殊要求：{{如有，比如要埋钩子、要留悬念等}}

请执行以下操作：

1. 读取大纲文件（如果提供了路径）
   - 理解大纲结构和每个模块的意图
   - 识别目标受众和平台特点

2. 生成完整初稿
   - 使用 [[90-系统配置/模板/文章模板]]
   - 严格遵循大纲结构，但允许合理展开
   - 根据平台特点调整：
     - 公众号：故事性、段落清晰、金句密度适中
     - 小红书：干货密度高、分点清晰、emoji适当、CTA强
     - X：简洁有力、适合串推、每点独立成推
     - 抖音脚本：口语化、节奏感、视觉提示

3. 保存到对应位置
   - 根据平台和内容类型选择正确的工厂文件夹
   - 文件名格式：日期-选题关键词-版本
   - 更新YAML frontmatter（word_count, status等）

4. 建立回链
   - 链接到大纲文件
   - 链接到选题文件
   - 链接到灵感来源

完成后告诉我：
- 生成的文章字数
- 保存的文件路径
- 核心内容亮点
- 建议的标题（3个备选）
```

## 简化版（快速使用）

```
帮我把这个大纲写成公众号文章，1500字左右，语气亲切。

【大纲】
[粘贴大纲]
```

## 示例

**输入：**
```
帮我把这个大纲写成小红书图文内容。

【大纲文件】03-内容工厂/1-大纲挑选区/2024-03-07-高敏感自救指南-大纲.md

要求：
- 适合小红书图文形式（1封面+5-7内页）
- 语气温暖、理解高敏感人群
- 每页一个核心观点+金句+行动建议
- 结尾强CTA
```

**Claude 会：**

1. 读取大纲文件，理解结构：
   - 封面：高敏感不是缺陷，是你的天赋
   - 模块1：什么是高敏感（不是玻璃心）
   - 模块2：高敏感的4大优势
   - 模块3：日常自救方法
   - 结尾：互动+关注引导

2. 生成小红书图文内容，输出结构：

```markdown
# 高敏感不是缺陷，是你的天赋｜4个自救方法

## 封面页
标题：高敏感不是缺陷，是你的天赋
副标题：4个自救方法｜高敏感人群必看
关键词：#高敏感 #自救指南 #心理成长

---

## 第1页：什么是高敏感？
**标题**：你不是玻璃心，只是高敏感

**内容**：
高敏感人群(HSP)约占人群15-20%
特点：深度加工信息、易被过度刺激、情绪反应强烈、感知细节敏锐

**金句**：
"敏感不是弱点，是你感知世界的天赋"

**行动**：
评论区扣1，让我知道有多少高敏感姐妹

---

## 第2-4页：...（继续生成）

## 结尾页
**CTA**：
收藏这篇，下次情绪崩溃时翻出来看
关注我，每天一个心理自救小技巧
下期预告：《高敏感如何在职场上发挥优势》

---
```

3. 保存到 `05-图文工厂/小红书/2024-03-07-高敏感自救指南-图文.md`

4. 建立回链到大纲和选题
