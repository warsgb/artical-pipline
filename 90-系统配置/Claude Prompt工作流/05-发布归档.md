# Claude Prompt：发布归档

## 使用场景
当你完成内容发布，需要归档文件、记录数据、生成复盘。

## Prompt 模板

```
@用Claude和Obsidian做内容生产工厂.md

帮我把这篇内容标记为已发布，完成归档和复盘。

【发布内容信息】
- 内容文件：{{原文路径或粘贴内容}}
- 发布平台：{{小红书/公众号/抖音/X等}}
- 发布链接：{{可选，发布后回填}}

【数据表现】（发布后填入）
- 发布时间：
- 阅读量/播放量：
- 点赞数：
- 收藏数：
- 评论数：
- 转发数：
- 转化率/其他：

请执行以下操作：

1. 移动归档
   - 将原文从内容工厂移动到 `06-已发布归档/{{平台名}}/`
   - 更新文件状态为「已发布」
   - 在YAML中添加发布日期和平台信息

2. 创建数据记录卡片
   - 在 `06-已发布归档/` 下创建同名数据记录文件
   - 使用数据记录模板，填入发布链接和数据
   - 关联到原文文件

3. 生成复盘卡片
   - 在 `06-已发布归档/` 下创建复盘文件
   - 使用 [[90-系统配置/模板/复盘模板]]
   - 分析亮点、不足、可复制的方法

4. 更新所有回链
   - 在原文中添加发布平台和复盘链接
   - 确保选题、大纲、灵感源都能追溯到发布结果

5. 生成下次优化建议
   - 基于数据表现，给出3条下次可优化的点
   - 推荐类似主题的新选题方向

完成后告诉我：
- 归档的文件路径
- 复盘卡片的核心结论（2-3句）
- 下次值得复用的方法
- 推荐尝试的新方向
```

## 简化版（快速使用）

```
帮我把这篇标记为已发布，生成复盘。

【内容】{{路径或粘贴}}
【平台】{{平台}}
【数据】阅读X，点赞Y，收藏Z
```

## 示例

**输入：**
```
帮我把这篇小红书图文标记为已发布，完成归档和复盘。

【内容文件】05-图文工厂/小红书/2024-03-07-高敏感自救指南-小红书.md
【平台】小红书
【发布链接】https://www.xiaohongshu.com/...

【数据表现】
- 发布时间：2024-03-08 09:30
- 阅读量：12,500
- 点赞数：1,200
- 收藏数：890
- 评论数：156
- 转发数：45
```

**Claude 会：**

1. 归档文件
   - 移动 `05-图文工厂/小红书/2024-03-07-高敏感自救指南-小红书.md`
   - 到 `06-已发布归档/小红书/2024-03-07-高敏感自救指南.md`
   - 更新YAML：status: 已发布, platform: 小红书, publish_date: 2024-03-08

2. 创建数据记录
   `06-已发布归档/小红书/2024-03-07-高敏感自救指南-数据.md`：
   ```yaml
   ---
   title: 高敏感自救指南 - 数据记录
   type: 数据记录
   platform: 小红书
   publish_date: 2024-03-08
   link: https://www.xiaohongshu.com/...
   ---

   ## 数据表现

   | 指标 | 数值 | 行业均值 | 表现 |
   |------|------|----------|------|
   | 阅读量 | 12,500 | 3,000 | 优秀 |
   | 点赞数 | 1,200 | 150 | 优秀 |
   | 收藏数 | 890 | 80 | 优秀 |
   | 评论数 | 156 | 20 | 优秀 |
   | 转发数 | 45 | 10 | 良好 |

   ## 数据分析

   ### 亮点
   - 收藏/点赞比高达74%，说明内容干货价值高，用户愿意留存
   - 评论互动率1.25%，远超行业均值，引发共鸣
   - "高敏感"话题精准击中目标人群（25-35岁女性）

   ### 可优化
   - 转发率偏低，结尾CTA可以更明确
   - 评论区可以主动引导话题讨论

   ---

   ## 关联笔记
   - 原文：[[06-已发布归档/小红书/2024-03-07-高敏感自救指南.md]]
   - 复盘：[[06-已发布归档/小红书/2024-03-07-高敏感自救指南-复盘.md]]
   ```

3. 生成复盘卡片
   `06-已发布归档/小红书/2024-03-07-高敏感自救指南-复盘.md`：
   - 总结亮点：干货密度高、情绪共鸣强、选题精准
   - 沉淀方法：选题策略、结构公式、金句技巧
   - 下次优化：结尾CTA、互动引导、系列化延伸

4. 返回总结：
   - "这篇表现优秀，收藏/点赞比74%说明干货价值极高。
   - 可复用的方法：痛点共鸣+干货密度+行动清单的结构公式。
   - 建议延伸系列：《高敏感职场生存指南》《高敏感人际边界建立》"
