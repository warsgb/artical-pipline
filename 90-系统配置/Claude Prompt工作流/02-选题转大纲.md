# Claude Prompt：选题转大纲

## 使用场景
当你想从选题池里挑选选题并开始撰写，让 Claude 帮你生成详细大纲。

## Prompt 模板

```
@用Claude和Obsidian做内容生产工厂.md

帮我从选题池里挑1个最值得写的选题，并生成详细大纲。

【要求】
- 平台：{{目标平台，如公众号/小红书/抖音}}
- 受众：{{目标受众}}
- 选题偏好：{{如果有特定方向，如实用干货/情感共鸣/观点输出}}

请执行以下操作：

1. 读取 `02-选题池/1-待写选题库/` 里的所有选题
   - 分析每个选题的潜力（热度、差异化、可操作性）
   - 推荐1个最值得写的，并说明理由

2. 为选中的选题生成详细大纲
   - 使用 [[90-系统配置/模板/大纲模板]]
   - 根据目标平台调整结构（公众号要故事线、小红书要干货密度、抖音要有钩子）
   - 大纲要具体到每个模块写什么、怎么开头、怎么过渡

3. 保存大纲到 `03-内容工厂/1-大纲挑选区/`
   - 文件名：日期+选题关键词+大纲
   - 更新YAML frontmatter

4. 更新选题状态
   - 把选中的选题从 `1-待写选题库/` 移动到 `2-已立项/`
   - 在大纲文件中添加回链到选题

完成后告诉我：
- 选了哪个选题
- 为什么选它
- 大纲的核心结构
```

## 简化版（快速使用）

```
从选题池给我挑1个选题，生成公众号大纲。

偏好：实用干货，受众是职场新人。
```

---

## 示例

**输入：**
```
从选题池给我挑1个选题，生成小红书图文大纲。

偏好：情感共鸣，女性成长主题。
```

**Claude 会：**
1. 扫描 `02-选题池/1-待写选题库/`，发现选题如：
   - 「30岁才明白的5个道理」
   - 「高敏感人群的自救指南」
   - 「为什么你总是讨好别人」

2. 分析后推荐「高敏感人群的自救指南」，理由：
   - 话题有热度（高敏感是近年热门标签）
   - 差异化空间大（可以从"优势"角度切入，而非"问题"角度）
   - 小红书受众匹配度高（女性用户占比高，情感成长类受欢迎）

3. 生成小红书图文大纲：
   - 封面标题：「高敏感不是缺陷，是你的天赋｜4个自救方法」
   - 正文结构：钩子（共鸣）+ 4个方法（每页1图+1金句）+ CTA

1. 保存到 `03-内容工厂/1-大纲挑选区/2026-03-07-高敏感自救指南-大纲.md`

2. 把选题移动到 `02-选题池/1-已立项/`
