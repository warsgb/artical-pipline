# 选题概述

## 核心观点
通过 OpenClaw 框架构建一个"商机发现 Agent"，可以自动化搜索用户痛点和抱怨，从中筛选出竞争少的蓝海商业机会。

## 目标受众
- **人群画像：**
  - 想创业但不知道方向的人
  - 产品经理想挖掘用户需求
  - 自由职业者想找服务方向
- **痛点/需求：**
  - 不知道如何发现商业机会
  - 手动搜索效率太低
  - 是否蓝难以判断机会海
- **预期收获：**
  - 学会使用 OpenClaw 构建 AI Agent
  - 掌握从用户痛点发现商机的思路
  - 获得可直接使用的 prompt 和操作步骤
