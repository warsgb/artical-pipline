# bo-x-pubwechat

微信公众号发布工具 - 使用 Chrome CDP 自动化发布 Markdown 文章到公众号草稿箱。

## 快速开始

```bash
# 基本用法
发布公众号：路径/到/文章.md

# 或
发布到微信：路径/到/文章.md

# 或在文件所在目录
bo-x-pubwechat 文章名-终稿.md
```

## 功能特点

- ✅ 自动提取标题、作者、摘要
- ✅ 自动转换 Markdown 为 HTML
- ✅ Chrome 浏览器自动化
- ✅ 保持登录状态
- ✅ 自动保存草稿
- ✅ 浏览器窗口保持打开，方便预览编辑

## 前提条件

1. Chrome 浏览器已安装
2. Bun 运行时可用
3. 首次使用需要扫码登录微信公众号

## 工作流程

```
Markdown文件 → 提取元数据 → 转换HTML → 启动Chrome → 登录检查 → 填充内容 → 保存草稿
```

## 常见问题

### Q: 首次使用会怎样？

A: 浏览器会自动打开，提示你扫码登录微信公众号。登录成功后会自动继续发布流程。

### Q: 文章保存在哪里？

A: 文章保存在微信公众号的草稿箱中，需要你手动点击"发布"按钮才能正式发布。

### Q: 可以添加封面图片吗？

A: Chrome CDP 方式暂不支持自动添加封面，需要在浏览器中手动添加。

### Q: 遇到依赖错误怎么办？

A: 本 skill 使用手动 HTML 创建方式，避免依赖问题，提高发布成功率。

## 技术架构

- **Chrome CDP**: 通过 Chrome DevTools Protocol 控制浏览器
- **Profile 隔离**: 使用独立的 Chrome profile，保持登录状态
- **HTML 转换**: 手动转换 Markdown 为 HTML，避免依赖问题
- **剪贴板粘贴**: 通过剪贴板 API 实现内容粘贴
