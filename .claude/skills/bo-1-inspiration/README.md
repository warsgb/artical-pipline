# bo-inspiration Skill

## 简介

`bo-inspiration` 是内容生产工厂的第一个阶段skill，用于将灵感素材自动入库并生成3个可写的选题。

## 功能特性

- ✅ 自动创建灵感卡
- ✅ 智能生成3个差异化选题
- ✅ 自动建立双向回链
- ✅ 使用标准化模板
- ✅ 支持多种触发方式

## 快速开始

### 基本用法

在Claude Code中说：

```
灵感入库：

[粘贴你的灵感内容]
```

### 示例

```
灵感入库：

刚看到一个视频说：「AI不会取代作家，但会用AI的作家会取代不会用AI的作家」
```

### 其他触发方式

你也可以说：
- "存进内容工厂：[内容]"
- "生成选题：[内容]"
- "把这个存起来并生成3个选题：[内容]"

## 工作流程

1. **创建灵感卡**
   - 位置：`01-灵感与素材库/1-日常灵感剪报/`
   - 格式：`YYYY-MM-DD-关键词.md`
   - 模板：灵感模板

2. **生成3个选题**
   - 位置：`02-选题池/待写选题库/`
   - 格式：`YYYY-MM-DD-选题标题.md`
   - 模板：选题模板
   - 每个选题有独特角度

3. **建立回链**
   - 灵感卡 → 3个选题
   - 每个选题 → 灵感卡
