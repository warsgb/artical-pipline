# bo-autoimages 技能

自动化生成配图系列 - 全自动化流程，零用户交互

## 触发条件

当用户说：
- "生成图片"
- "生成配图"
- "bo-autoimages"
- 或提供内容并要求生成配图

## 功能特点

- **完全自动化**：无用户交互，从内容到图片一键生成
- **智能分析**：自动识别内容类型（教程/推荐/知识/避坑/效率/故事）、提取关键点
- **自动数量判断**：基于内容字数和结构，生成 3-6 张图片
- **智能比例判断**：根据目标平台自动选择 3:4（小红书）或 4:3（公众号）
- **风格适配**：基于内容类型智能推荐 notion/bold/cute/fresh/minimal/warm 风格
- **高质量输出**：2k质量，PNG格式

## 使用方式

```bash
# 方式1: 直接提供内容
/bo-autoimages --content "5个AI工具推荐：ChatGPT、Midjourney、Claude..."

# 方式2: 从文件读取
/bo-autoimages --file ./articles/my-article.md

# 方式3: 简化调用
/bo-autoimages "内容文本"
```

## 可选参数

- `--ar, --aspect-ratio`: 图片比例（3:4 或 4:3，默认自动判断）
- `--output, -o`: 输出目录（默认：05-图文工厂/img-{主题名}）
- `--style, -s`: 强制指定风格（notion/bold/cute/fresh/minimal/warm）
- `--image-count, -n`: 强制指定图片数量（3-6）
- `--provider, -p`: 图片生成服务（google/openai/replicate/dashscope）

## 智能特性

### 1. 图片比例自动判断

| 线索来源 | 判断结果 |
|---------|---------|
| 素材中 `platform: 小红书` / `platform: XHS` | 3:4 |
| 素材中 `platform: 公众号` / `platform: 微信` | 4:3 |
| 用户说"小红书配图" | 3:4 |
| 用户说"公众号配图" / "微信配图" | 4:3 |
| 用户说"抖音" / "短视频" | 9:16 |
| 无明确线索 | 默认 3:4（小红书） |

### 2. 图片数量判断

| 字数范围 | 图片数量 | 适用场景 |
|---------|---------|---------|
| <300字 | 3张 | 简短分享 |
| 300-800字 | 4张 | 标准内容 |
| 800-1500字 | 5张 | 详细教程 |
| >1500字 | 6张 | 深度长文 |

### 3. 风格推荐

| 内容类型 | 推荐风格 | 特点 |
|---------|---------|------|
| 教程/指南 | notion | 高信息密度，知识感强 |
| 推荐/好物 | cute/fresh | 清新可爱，种草力强 |
| 知识科普 | notion | 专业严谨，易读性强 |
| 避坑/警示 | bold | 醒目有力，警示性强 |
| 效率/职场 | minimal | 简洁高效，专业感强 |
| 温暖/故事 | warm | 情感共鸣，亲和力强 |

### 4. 布局策略

- **P1封面**: Sparse布局，突出标题，吸引注意力
- **P2核心**: Dense布局，核心观点，高信息密度
- **P3-P(N-1)**: 根据内容类型选择 list/comparison/flow 布局
- **P结尾**: Sparse布局，CTA引导，互动性强

## 技术实现

调用 baoyu-image-gen 技能进行图片生成：
- 使用 P1 作为后续图片的参考图，保证视觉一致性
- 支持 Google/OpenAI/Replicate/DashScope 多个 provider
- 输出格式：PNG，2k 质量

**提示词要求：**
- **强制中文输出**：在所有图片生成提示词中必须强调"**只输出中文文字，不要出现任何英文**"
- 提示词模板示例：`{内容描述}，Notion风格，极简设计，**图片中的所有文字必须使用中文，不要出现任何英文**`
- 确保图片中的标题、关键词、列表项等所有文本内容都是中文

## 与 baoyu-xhs-images 的区别

| 特性 | bo-autoimages | baoyu-xhs-images |
|------|---------------|------------------|
| 用户交互 | 无（全自动） | 有（多步确认） |
| 默认风格 | Notion | 可选（10种） |
| 比例选择 | 自动判断 3:4/4:3/9:16 | 需手动指定 |
| 默认数量 | 4张（智能判断3-6） | 1-10张可选 |
| 使用场景 | 快速批量生产 | 精细化定制 |

## 示例

```bash
# 自动判断（根据素材中的platform字段）
/bo-autoimages @02-选题池/待写选题库/2026-03-08-xxx.md

# 指定小红书比例
/bo-autoimages --ar 3:4 "内容..."

# 指定公众号比例
/bo-autoimages --ar 4:3 "内容..."

# 指定图片数量
/bo-autoimages --image-count 6 "长内容..."
```

## 注意事项

1. **内容语言要求**：
   - 生成图片时，提示词必须强调"**只输出中文文字，不要出现任何英文**"
   - 所有图片中的标题、关键词、列表项等文本必须是中文
   - 避免图片中出现英文单词、缩写或中英文混杂

2. 内容长度建议 100-2000 字
3. 输出目录固定为 05-图文工厂，文件夹以 `img-` 开头
4. 图片生成速度取决于 provider 和图片数量
5. 建议使用 google provider 获得最佳质量和速度平衡
